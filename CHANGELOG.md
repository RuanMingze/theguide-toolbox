# 更新日志 (CHANGELOG)

## [2026-03-28] - 重大功能更新

### 新增功能
- **设置页面**：新增完整的设置页面，支持自定义壁纸、毛玻璃效果、图片上传等功能
- **登录系统**：集成 GitHub OAuth 登录功能，支持用户认证和个性化欢迎
- **节日日历**：日历组件支持显示节日标记，鼠标悬停查看节日介绍
- **节日通知**：页面加载时自动检测节日，右上角弹出节日祝福通知条
- **PWA 支持**：添加渐进式 Web 应用支持，可安装到桌面
- **自定义提示框**：替换所有浏览器默认 alert 为美观的 toast 提示
- **英文支持**：新增英文翻译，支持英文用户使用

### 功能优化
- **时钟组件**：修复水合错误，优化服务器/客户端渲染一致性
- **导航栏**：优化导航栏样式和用户体验
- **登录流程**：完善登录回调和错误处理
- **壁纸系统**：修复默认壁纸显示问题，支持实时预览和保存

### 技术改进
- 添加 Edge Runtime 支持
- 优化 React Suspense 边界处理
- 添加环境变量重建配置
- 完善 TypeScript 类型定义
